FolioOrb runs entirely on your machine — no account, no cloud. Download the desktop app for
macOS or Windows below (no Python, no terminal), or [run it in your browser as a local web
app from source](/FolioOrb/build-from-source/) on any platform with Python 3.11+.

The buttons above always resolve to the newest **stable** release. You can also browse
[every release](https://github.com/udhawan97/FolioOrb/releases) or use the
[landing page](https://udhawan97.github.io/FolioOrb/) with platform-aware download buttons.

## Release channels

| Channel | Where it comes from | Who it's for |
| --- | --- | --- |
| **Stable** | A version tag (`v5.6.0`), cut manually | Everyone. This is the default download. |
| **latest-main** | Refreshed automatically on every merge to `main` | Testing the newest changes; may be unstable. |

The `latest-main` prerelease is offered under **Development builds** on the website and in
the [releases list](https://github.com/udhawan97/FolioOrb/releases). It always holds a
single, newest build straight from `main`.

<Aside type="caution" title="Early builds are unsigned">
FolioOrb doesn't have a paid Apple or Windows code-signing certificate yet, so macOS
and Windows will show a warning the first time you open it. This is expected for an
open-source app. The [macOS](/FolioOrb/install-macos/) and
[Windows](/FolioOrb/install-windows/) guides show exactly what you'll see and how to
proceed. See [Releases &amp; versioning](/FolioOrb/releases-and-versioning/) for the
signing roadmap.
</Aside>

## Verify your download

Every release includes a `SHA256SUMS.txt`. After downloading, confirm your file matches:

```bash
# macOS / Linux — run in the folder holding the download + SHA256SUMS.txt
shasum -a 256 -c SHA256SUMS.txt
```

```powershell
# Windows PowerShell — compare against the line in SHA256SUMS.txt
Get-FileHash .\FolioOrb-Windows-x64-v5.6.0-Setup.exe -Algorithm SHA256
```

A match confirms the file is byte-for-byte what CI published to GitHub Releases.
