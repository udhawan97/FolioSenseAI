title: Installation
description: Ways to install FolioSenseAI — the desktop app, or from source.
---

There are two ways to get FolioSenseAI:

- **Desktop app (recommended)** — a native macOS or Windows download, no Python required.
  See [Download](../../download/), then [Install on macOS](../../install-macos/) or
  [Install on Windows](../../install-windows/).
- **From source** — run it with Python 3.11+ on any platform (including Intel Macs and
  Linux), or build the installers yourself. See [Build from source](../../build-from-source/).

Already installed? See [Updating](../../updating/) to move to a newer version without losing
your data.
