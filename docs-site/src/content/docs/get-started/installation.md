---
title: Installation
description: Ways to run FolioOrb — in your browser as a web app, or as a desktop app for macOS or Windows.
---

There are two ways to run FolioOrb — it's the same dashboard either way, and both run entirely
on your own machine:

- **As a web app (from source)** — run it with Python 3.11+ and use it in your browser at
  `localhost:8000`, on macOS, Windows, or Linux (including Intel Macs). See
  [Build from source](../../build-from-source/).
- **As a desktop app** — a one-click macOS or Windows download that runs the same dashboard in
  its own window, with no Python required. See [Download](../../download/), then
  [Install on macOS](../../install-macos/) or [Install on Windows](../../install-windows/).

Already installed? See [Updating](../../updating/) to move to a newer version without losing
your data.
