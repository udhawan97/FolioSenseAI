The Windows download is a per-user installer (`.exe`) for **64-bit Windows 10 and 11**. It
installs without administrator rights and registers a clean uninstaller.

<Steps>

1. Download `FolioSenseAI-Windows-x64-v4.3.2-Setup.exe` from the
   [latest release](https://github.com/udhawan97/FolioSenseAI/releases/latest).

2. Run it. If prompted about the WebView2 runtime, let it install — it's a small Microsoft
   component the app uses to render its window (already present on Windows 11).

3. Finish the wizard and launch FolioSenseAI from the Start menu.

</Steps>

## First launch: the SmartScreen warning

Because early builds aren't code-signed, Windows SmartScreen may show
*"Windows protected your PC."* This is expected for a new open-source app. To proceed:

<Steps>

1. Click **More info**.

2. Click **Run anyway**.

</Steps>

<Aside type="note" title="Why the warning?">
SmartScreen flags installers it hasn't seen build reputation for yet. It doesn't mean the
file is unsafe. You can [verify your download](/FolioSenseAI/download/#verify-your-download)
against the published `SHA256SUMS.txt`. Windows code signing is on the
[roadmap](/FolioSenseAI/releases-and-versioning/#code-signing-roadmap).
</Aside>

## Where your data lives

| What | Location |
| --- | --- |
| Program files | `%LOCALAPPDATA%\Programs\FolioSenseAI` |
| Portfolio database | `%APPDATA%\FolioSenseAI\database\` |
| Config (`.env`) | `%APPDATA%\FolioSenseAI\.env` |

Your data sits outside the program folder, so updating or reinstalling never touches it.

## Uninstall

Open **Settings → Apps → Installed apps**, find **FolioSenseAI**, and choose **Uninstall**.
To also remove your data, delete `%APPDATA%\FolioSenseAI`.
