The macOS download is a standard `.dmg` for **Apple Silicon** (M1 and newer), targeting
macOS 11 Big Sur and later.

<Steps>

1. Download `FolioSenseAI-macOS-arm64-v4.3.4.dmg` from the
   [latest release](https://github.com/udhawan97/FolioSenseAI/releases/latest).

2. Open the `.dmg` and drag **FolioSenseAI** into **Applications**.

3. Launch it from Applications or Spotlight.

</Steps>

## First launch: the Gatekeeper warning

Because early builds aren't notarized, macOS blocks the first launch with a message like
*"FolioSenseAI can't be opened because Apple cannot check it for malicious software."*
This is expected. To allow it:

<Steps>

1. Try to open the app once (double-click). Dismiss the warning dialog.

2. Open **System Settings → Privacy &amp; Security**.

3. Scroll down to the message about FolioSenseAI being blocked and click **Open Anyway**.

4. Confirm with **Open**. macOS remembers the choice — future launches are one click.

</Steps>

<Aside type="note" title="Why the warning?">
It means the app isn't signed with a paid Apple Developer certificate — not that anything
is wrong with it. You can [verify your download](/FolioSenseAI/download/#verify-your-download)
against the published `SHA256SUMS.txt` for extra confidence. Signing and notarization are on
the [roadmap](/FolioSenseAI/releases-and-versioning/#code-signing-roadmap).
</Aside>

## Where your data lives

| What | Location |
| --- | --- |
| Portfolio database | `~/Library/Application Support/FolioSenseAI/database/` |
| Config (`.env`) | `~/Library/Application Support/FolioSenseAI/.env` |

The app never writes inside `/Applications`, so updates and reinstalls leave your data
untouched. See [Updating](/FolioSenseAI/updating/).

## Uninstall

Drag **FolioSenseAI** from Applications to the Trash. To also remove your data, delete
`~/Library/Application Support/FolioSenseAI`.

## Intel Macs

An Intel-native download isn't published yet. On an Intel Mac, use the
[install script or build from source](/FolioSenseAI/build-from-source/) — both run fine
with Python 3.11+.
